# Learn Django

This is a simple Django project created to learn the basics of the Django framework.

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* Python 3.x
* pip

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/learn_django.git
   ```
2. Install dependencies
   ```sh
   pip install -r requirements.txt
   ```

### Running the application

1. Apply migrations
   ```sh
   python manage.py migrate
   ```
2. Run the development server
   ```sh
   python manage.py runserver
   ```
   The application will be available at http://127.0.0.1:8000/.

## About the Polls App

This project includes a simple `polls` application. This application allows users to vote on questions. It is based on the official Django tutorial.
